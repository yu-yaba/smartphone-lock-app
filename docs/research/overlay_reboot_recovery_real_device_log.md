### 実行メモ
- power_off_boot:
  - ロック開始 → **電源OFF** → 電源ON → 起動完了 → 復帰確認
- lock_60m / 90m / 120m:
  - ロック開始 → 実時間待機 → **再起動（または電源OFF→起動）** → 復帰確認
- battery_saver_long:
  - ロック開始 → 省電力ON → 実時間待機 → 再起動 → 復帰確認
- screen_off_long:
  - ロック開始 → 画面OFF放置 → 実時間待機 → ロック継続確認（再起動なし）
- unlock_transition:
  - ロック開始 → **端末再起動** → 起動画面でロック維持 → **ユーザー解錠** → CE/DP 切替後もオーバーレイ維持を確認
- unlock_transition_early:
  - 再起動後、**起動直後（起動画面が出たら即）**に解錠 → CE/DP 切替後の維持確認
- unlock_transition_late:
  - 再起動後、**起動画面のまま3〜5分待機** → その後解錠して維持確認
- battery_opt_*:
  - 端末設定で最適化ポリシーを切替（制限 / 最適化 / 制限なし）
  - ロック開始 → 再起動 → 復帰/心拍ログを確認
